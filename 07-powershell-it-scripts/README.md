# Project 07 - PowerShell IT Scripts

## Status
Completed - March 2026

## Overview
Built a collection of PowerShell automation scripts 
for IT administration tasks including bulk user 
creation, system health reporting, and inactive 
user auditing.

## Scripts Created

### Script 1 - Bulk AD User Creation
- Reads user details from a defined array
- Creates users in correct Active Directory OUs
- Sets passwords and enables accounts automatically
- Reports success or failure for each user

### Script 2 - System Health Report
- Checks CPU load percentage
- Reports total, used and free RAM
- Reports disk usage on C drive
- Lists stopped automatic services
- Saves timestamped report to C drive

### Script 3 - Inactive User Audit
- Finds AD users inactive for 90 or more days
- Exports results to a CSV report
- Shows username, department and last logon date

## Tools Used
- Windows PowerShell ISE
- Active Directory PowerShell Module
- Windows Server 2022

## Skills Demonstrated
- PowerShell scripting
- Active Directory automation
- System monitoring and reporting
- User account auditing
- Error handling in scripts
