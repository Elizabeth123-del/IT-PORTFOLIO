# Project 04 — Active Directory Lab

## Status
Completed — March 2026

## Overview
Configured Windows Server 2022 as a fully functional 
Domain Controller with Active Directory Domain Services 
for a fictional company domain corp.local

## What I Built

### Domain Controller
- Server renamed to DC01
- Promoted to Domain Controller
- Domain: corp.local
- DNS configured automatically

### Organizational Units
- HR
- Finance
- IT
- Management

### User Accounts
- John.Smith (HR)
- Mariam.Damilola (HR)
- sarah.johnson (Finance)
- Tim.Thomas (IT)
- elizabeth.smith (Management)

### Group Policy Objects
- Screensaver-Lock-Policy (linked to corp.local)
- Disable-USB-Storage-Finance (linked to Finance OU)
- Password-Policy (linked to corp.local)

## Tools Used
- VMware Workstation Pro
- Windows Server 2022
- Active Directory Users and Computers
- Group Policy Management Console

## Skills Demonstrated
- Active Directory installation and configuration
- Domain Controller promotion
- Organizational Unit design
- User account management
- Group Policy Object creation and linking
- DNS configuration
