# Project 06 - VLAN Network Design

## Status
Completed - March 2026

## Overview
Designed and configured a segmented multi-department 
network using VLANs, inter-VLAN routing, and Access 
Control Lists in Cisco Packet Tracer.

## Network Design

### VLANs Created
- VLAN 10 - HR (192.168.10.0/24)
- VLAN 20 - Finance (192.168.20.0/24)
- VLAN 30 - IT (192.168.30.0/24)
- VLAN 40 - Management (192.168.40.0/24)

### Devices Used
- Cisco 2911 Router
- Cisco 3560 Layer 3 Switch
- 4 PCs (one per department)

### Configuration Completed
- VLANs created and named on switch
- Ports assigned to correct VLANs
- Trunk port configured between switch and router
- Router on a stick configured with sub-interfaces
- Inter-VLAN routing verified with ping tests
- ACL configured to block Finance from accessing IT
- Management VLAN can reach all other VLANs

## Tools Used
- Cisco Packet Tracer
- draw.io

## Skills Demonstrated
- VLAN creation and configuration
- Inter-VLAN routing
- Trunk port configuration
- Access Control List implementation
- Network topology design
- Network documentation
