#  VMware Home Lab Environment Setup

##  Overview
This project demonstrates the deployment of a virtualized home lab environment using VMware Workstation to simulate real-world system administration and networking scenarios.

---

## Objective
- Build a virtual lab environment for hands-on IT practice  
- Install and configure multiple operating systems  
- Prepare infrastructure for Active Directory and networking projects  

---

## Technologies Used
- VMware Workstation  
- Windows Server 2022  
- Ubuntu Server  
- Virtual Networking (NAT & Host-Only)  

---

## Implementation

- Installed VMware Workstation and configured virtualization settings  
- Created and provisioned multiple virtual machines  
- Installed Windows Server 2022 and Ubuntu Server operating systems  
- Configured NAT for internet access and Host-Only network for internal communication  
- Verified connectivity between virtual machines using ping tests  
- Created snapshots for system recovery and rollback  

---

## Network Configuration

- Configured NAT adapter for external internet access  
- Configured Host-Only adapter for isolated internal lab communication  
- Tested connectivity between VMs to ensure proper network setup  

---

##  Challenges & Solutions

- Encountered connectivity issues between virtual machines  
- Identified incorrect network adapter configuration  
- Resolved by properly configuring NAT and Host-Only adapters  
- Verified fix using successful ping tests
